## 📃 프로젝트 개요

### 편의점 자동 진열·판매 로봇 (Albaro)

편의점 환경에서 반복적으로 발생하는 진열 보충 및 재고 관리 문제를 해결하고
무인 편의점의 취약점인 연령 제한 상품 관리까지 고려하여 실제 매장 적용 가능한 자동화 로봇

#### 📆 개발 기간 : 2025년 12월 22일 ~ 2026년 1월 06일

#### 📜 관련 문서 노션 링크: [노션링크](https://kaput-promise-bd0.notion.site/2-2d1f99f5d393802c99a0de7b53bf58cf?source=copy_link)

## 🛠️ 평가 및 피드백

### 완성도 평가

- 단순 기능 시연이 아닌 실제 매장 시나리오 기반 동작 흐름을 완성

### 추후 개선점 및 보완할 점

- 다양한 환경 데이터를 추가 학습하여 객체 인식 정확도 및 안정성 향상 필요
- 연령 판별 결과에 대한 다단계 검증 로직 추가

### 우리 팀이 잘한 부분 / 아쉬운 점

- Wake-up Word 분기 구조를 통한 기능 제어
- 연령 판별 모델의 특성상 정확도에 한계가 존재
- 진열장의 음료 수량을 세세하게 구분하지 못함

### 느낌 점 및 경험한 성과

- 객체 인식, 로봇 제어, 음성 처리 등 서로 다른 기술 을 ROS2 기반의 하나의 시스템 통합 경험
- 음성 제어 기능을 직접 구현하며, 음성 인식 기술의 동작 원리와 적용 방식에 대한 이해도 향상
